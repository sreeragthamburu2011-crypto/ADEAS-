BUTTON 1: CONNECT BLUETOOTH
═══════════════════════════════════════════════════════════════════════════════

Component: ListPicker_BT
Text: "Select HC-05"
Properties: Blue background, White text

Blocks:
When Screen1.Initialize
  Call ListPicker_BT.Open

When ListPicker_BT.AfterPicking
  Set BluetoothClient1.Address to ListPicker_BT.Selection
  Call BluetoothClient1.Connect

When BluetoothClient1.Connected
  Set Label_Status.Text to "Connected to Car"
  Set Label_Status.BackgroundColor to Green

═══════════════════════════════════════════════════════════════════════════════
BLOCK 12: RECEIVE DATA FROM ARDUINO
═══════════════════════════════════════════════════════════════════════════════

Component: BluetoothClient1

Blocks:
When BluetoothClient1.ReceiveText numberOfBytes -1
  set message to call BluetoothClient1.ReceiveText numberOfBytes -1
  
  If message = "ACCIDENT_DETECTED"
    Then
      Set Label_Status.Text to "ACCIDENT! CAR STOPPED"
      Set Label_Status.BackgroundColor to Red
      Call Notifier1.ShowAlert title "ACCIDENT DETECTED" 
                                message "Press RESET to recover"
  
  Else If message = "SYSTEM_RESET_ACK"
    Then
      Set Label_Status.Text to "System Ready"
      Set Label_Status.BackgroundColor to Green

═══════════════════════════════════════════════════════════════════════════════
COMMAND REFERENCE
═══════════════════════════════════════════════════════════════════════════════

F = Forward (TouchDown sends F, TouchUp sends S)
B = Backward (TouchDown sends B, TouchUp sends S)
L = Left (TouchDown sends L, TouchUp sends S)
R = Right (TouchDown sends R, TouchUp sends S)
S = Stop
r = Reset System
W = LED ON
w = LED OFF
V = Buzzer ON
v = Buzzer OFF

═══════════════════════════════════════════════════════════════════════════════
MESSAGES FROM ARDUINO
═══════════════════════════════════════════════════════════════════════════════

ACCIDENT_DETECTED - Collision detected, car stopped
SYSTEM_RESET_ACK - System reset successful
EMERGENCY_AUTO_ALERT - 20 second timeout reached
ADEAS_READY - Arduino online and ready
LED_ON - LED confirmed on
LED_OFF - LED confirmed off
BUZZER_ON - Buzzer confirmed on
BUZZER_OFF - Buzzer confirmed off
